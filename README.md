# 🩺 Skin Companion AI

An AI-powered skin analysis web application that helps users analyze skin conditions from uploaded images and provides personalized skincare insights using Google's Gemini AI.

🔗 **Live Demo:** https://skin-companion-ai-753848538466.asia-southeast1.run.app

---

## 📌 Features

- 🔐 User Authentication (Register/Login)
- 📧 Email OTP Verification
- 🔑 Forgot Password with OTP
- 📸 Capture image using Camera
- 🖼️ Upload image from Gallery
- 🤖 AI-powered Skin Analysis using Gemini AI
- 📊 Detailed Skin Health Report
- 💡 Personalized Skincare Recommendations
- 📱 Responsive UI (Desktop & Mobile)
- 🌐 Cloud Deployed Application

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Rajsinha7/skin-companion-ai.git

cd skin-companion-ai
```

### Install Dependencies

```bash
npm install
```

### Create Environment File

Create a `.env.local` file and add:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### Run Development Server

```bash
npm run dev
```
